# E-Sign Virginia

Professional Next.js legal-tech platform for Virginia electronic signature and online appointment workflow.

## Included
- Public marketing homepage
- Customer login/dashboard
- E-Notary login/dashboard
- Appointment request system
- Responsive Tailwind styling
- Reusable component architecture

## Run Locally
```bash
npm install
npm run dev
```

## Deploy
Recommended deployment on Vercel.
